## 字符型常量和字符串常量

1. **含义**：字符型常量相当于一个整型值（ASCII值）可以参加表达式运算；字符串常量：表示一个地址（内存中的位置）
2. **内存大小**：字符型常量占两个字节，字符串常量占若干个字节

char 的包装类 `**Character**` 成员变量 `**Character.SIZE = 16**` 单位 `**bits**` ，除以8 (`**1byte = 8bits**`) 得到 2字节

| No.  | 数据类型         | 大小/位<bits> | 可表示数据范围                           | 默认值   | 包装类    |
| ---- | ---------------- | ------------- | ---------------------------------------- | -------- | --------- |
| 1    | byte（字节型）   | 8             | -128~127                                 | 0        | Byte      |
| 2    | short（短整型）  | 16            | -32768~32767                             | 0        | Short     |
| 3    | int（整型）      | 32            | -2147483648~2147483647                   | 0        | Integer   |
| 4    | long（长整型）   | 64            | -9223372036854775808~9223372036854775807 | 0        | Long      |
| 5    | float（单精度）  | 32            | -3.4E38~3.4E38                           | 0.0      | Float     |
| 6    | double（双精度） | 64            | -1.7E308~1.7E308                         | 0.0      | Double    |
| 7    | char（字符）     | 16            | 0~255                                    | '\u0000' | Character |
| 8    | boolean（布尔）  | -             | true或false                              | false    | Boolean   |

## 基本类型和引用类型

### 存储位置

#### 基本变量类型

- 在方法中定义的非全局基本数据类型变量的具体内容是存储在栈中的

#### 引用变量类型

- 只要是引用数据类型变量，其具体内容都是存放在堆中的，而栈中存放的是其具体内容所在内存的地址
  *ps:通过变量地址可以找到变量的具体内容，就如同通过房间号可以找到房间一般*



```java
public class Main{
   public static void main(String[] args){
       //基本数据类型
       int i=1;
       double d=1.2;
       
       //引用数据类型
       String str="helloworld";
   }
}
```

## 自增、自减

符号在前就先加/减，符号在后就后加/减


```java
public static void main(String[] args) {

        //符号在前就先加/减，符号在后就后加/减
        //赋值
        int a = 1;
        int b = 10;
        ++a;//自增 运算 ++a =a+1  a++ =a+1
        System.out.println(a);//2
        a++;//自增 运算
        System.out.println(a);//3
        b = ++a;//自增  赋值 b = (a+1);
        System.out.println(b);//4
        b = a++;//自增  赋值 b = a; a= a+1;
        System.out.println(b);//4
        System.out.println(a);//5

}
```

## continue、break、和return 的区别

循环结构体，当循环条件不满足或者循环次数达到要求是，循环会正常结束，有时需要自定义。

1. continue: 跳出当前循环，继续下一次循环
2. break：跳出整个循环，继续执行循环下面的语句

1. return：跳出整个方法，结束方法运行

1. 1. return  ：直接使用返回方法，没有返回值函数方法
   2. return value：有返回值函数方法
